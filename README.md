# fake-job-postings-classifier
NLP and ML project to detect fake job postings using TF-IDF vectorization and Logistic Regression with 90%+ accuracy.

Dataset used: Fake Job Posting Prediction (Kaggle)
Download here: https://www.kaggle.com/datasets/shivamb/real-or-fake-fake-jobposting-prediction
Place file in /data before running notebook.
